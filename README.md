# HDDS Agentic Clinical Intelligence Platform

A full-stack clinical intelligence prototype using synthetic data, AI agents, FastAPI, and React.

> **All outputs are for prototype/demo purposes only — intended for clinician review, not a diagnosis.**

## Quick Start

```bash
git clone https://github.com/Kai23-dev/HDDS_Agentic_Clinical_Intelligence_Platform.git
cd HDDS_Agentic_Clinical_Intelligence_Platform/hdds_clinical_intelligence

pip install -r requirements.txt       # Python deps
python run_hdds_prototype.py --all    # Generate insights

# Terminal 1: Start backend
python api.py

# Terminal 2: Start frontend
cd frontend && npm install && npm run dev
```

Open **http://localhost:5173** in your browser.

See [hdds_clinical_intelligence/README.md](hdds_clinical_intelligence/README.md) for full documentation, troubleshooting, and project structure.

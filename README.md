# HDDS Agentic Clinical Intelligence Platform

> A clinical intelligence prototype that uses synthetic healthcare data, extraction/harmonization, and AI agents to generate clinician-reviewable medical insights.

## Quick Start

```bash
cd hdds_clinical_intelligence
python run_hdds_prototype.py          # Single patient
python run_hdds_prototype.py --all    # All 3 patients
```

See [hdds_clinical_intelligence/README.md](hdds_clinical_intelligence/README.md) for full documentation.
